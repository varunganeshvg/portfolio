Library Management Portal

A role-based library management system designed to control access to educational resources using authentication, authorization, and structured backend services.

Live Project

Authentication & Role Management

The application uses a centralized authentication mechanism where users log in as one of three roles: Admin, Staff, or Student. Upon successful login, a token is generated and used to authorize access throughout the application.

Admin Module

The Admin has complete control over the system. After logging in, the Admin can:

  • Create and manage staff accounts
  • Assign staff to specific classes and subjects
  • View all registered students and staff
  • Control how educational resources are distributed

System Architecture

The system follows a service-oriented approach with clear separation of concerns:

  • Auth Service: Handles login, registration, token creation, and validation
  • Book Service: Manages book uploads, class mapping, and access control

This separation improves scalability, security, and maintainability of the application.
